# Privacy Policy / プライバシーポリシー

**Wakarumade**

**Last updated: 2026-03-05**

### 日本語　　

### 1. 概要

Wakarumade は、小学生向けの算数問題の理解をサポートする学習アプリです。

当アプリはユーザーのプライバシーを尊重し、サービス提供に必要な最小限の情報を取り扱います。

### 2. 取り扱う情報

当アプリは以下の情報を取り扱う場合があります。

**(1) OCRで抽出されたテキスト**

ユーザーが撮影または選択した画像は、端末上でOCR（文字認識）されます。

画像データ自体はサーバーへ送信しません。

OCRで抽出されたテキストは、問題解析や学習支援のためにサーバーへ送信される場合があります。

**(2) 会話データ**

ユーザーとAIチューターとのチャット内容は、学習履歴の提供およびサービス改善のために保存される場合があります。

**(3) 匿名識別子**

アプリ利用管理のため、以下の匿名IDを生成・利用する場合があります。

- guestId
- userId
これらは単独で個人を特定する情報ではありません。

### 3. 音声入力について

当アプリは音声入力機能を提供します。

音声認識には Apple の音声認識機能を利用し、利用環境によっては Apple のサーバー処理を含む場合があります。

当アプリのサーバーへは、必要に応じてテキスト化後の内容のみ送信されます。

### 4. 利用目的

取り扱う情報は、以下の目的のためにのみ利用します。

- 算数問題の解析
- 学習サポート機能の提供
- 利用状況・利用制限の管理
- サービス改善

### 5. 第三者提供・外部委託

当アプリは、法令に基づく場合を除き、ユーザーデータを第三者へ販売しません。

ただし、サービス提供に必要な範囲で、クラウド基盤・データベース・AI推論基盤等の外部事業者へ処理を委託する場合があります。

### 6. データ保存

一部の学習履歴データは、学習体験の提供とサービス改善のためにサーバーへ保存される場合があります。

保存期間は、運用上必要な期間または法令で求められる期間とします。

### 7. AIによる処理について

本アプリでは、算数の問題に対するヒントや説明を生成するために、AI処理サービスを使用しています。

その際、以下のデータがAI処理サービスに送信される場合があります。

- OCRによって抽出された算数の問題文
- 学習中にユーザーが入力したチャットメッセージ
- 匿名のユーザー識別子

なお、**画像データそのものは端末上で処理され、サーバーには送信されません。**

これらのデータは、ヒントや説明を生成する目的のみに使用されます。

広告、トラッキング、マーケティング目的で使用されることはありません。

### English

### 1. Overview

Wakarumade is an educational app designed to help elementary school students understand math word problems.

We respect user privacy and process only the minimum data necessary to provide the service.

### 2. Information We Process

The app may process the following data:

**(1) OCR-Extracted Text**

Images captured or selected by the user are processed on-device using OCR.

Image files themselves are not sent to our server.

Extracted text may be sent to the server to analyze problems and provide tutoring assistance.

**(2) Chat Data**

Messages between the user and the AI tutor may be stored to provide learning history and improve the service.

**(3) Anonymous Identifiers**

The app may generate and use anonymous identifiers such as:

- guestId
- userId
These identifiers do not, by themselves, identify a specific individual.

### 3. Voice Input

The app provides voice input functionality.

Speech recognition uses Apple’s speech recognition services, which may involve Apple server-side processing depending on the environment.

Our server receives only text (not raw audio), when needed.

### 4. Purpose of Use

Processed data is used only for:

- Analyzing math problems
- Providing tutoring support
- Managing usage and usage limits
- Improving the service

### 5. Third-Party Sharing and Processors

We do not sell user data to third parties, except as required by law.

However, we may use third-party processors (e.g., cloud infrastructure, database services, and AI inference providers) as necessary to operate the service.

### 6. Data Retention

Some learning history data may be stored on servers to support the learning experience and improve the service.

Retention periods are based on operational necessity and legal requirements.

### 7. AI Processing

To generate hints and explanations, the app sends the extracted math problem text and user chat messages to an AI processing service.

The following data may be transmitted:
• Extracted problem text
• User chat messages
• Anonymous user identifier

Images are processed on-device and are not transmitted.

The data is used solely to generate responses and is not used for advertising, tracking, or marketing purposes.
